# Telegram бот-агрегатор новостей "MyPikabu"

## Описание проекта
"MyPikabu" - бот, с помощью которого можно:
    1. Настроить парамтеры показа новостей:
        - категория
        - частота рассылки
        - источники новостей
    2. Просматривать новости с выбранными параметрами
    3. Фильтровать новости по дате, релевантности и другим параметрам
    4. Автоматически сохранять историю просмотренных новостей

## Команды (возможности)
/start - запуск бота
/subscribes - настройка парметров показа новостей
/news - просмотр новостей

## Инструкция по запуску
1. Клонировать проект
2. Установить зависимости `npm install`
3. Создать `.env` файл: `BOT_TOKEN=ваш токен` и указать свой токен
4. Запустить бота `npm start`
